# Privacy and Security Checklist

## Privacy
- [ ] Data minimization applied.
- [ ] Sensitive attributes identified.
- [ ] Access controls in place.
- [ ] Retention/deletion plan defined.

## Security
- [ ] Secrets managed outside source code.
- [ ] Third-party dependencies reviewed.
- [ ] Prompt injection or adversarial input checks run.
- [ ] Output filtering policy defined.

## Operational
- [ ] Incident owner assigned.
- [ ] Escalation contacts listed.
- [ ] Backup/rollback artifact available.
